# smallpy
Small Python implement with C++, including simple Python object, add, print methods.

### compile 
```
$ cd src
$ g++ *.cpp -o smpy
```
### run and test
```
$ ./smpy 
********** Python Research **********
>>> a = 2
>>> b = 4
>>> c = a + b
>>> print c
6
>>> d = "hello world" 
>>> print d
hello world
>>> exit
```
